<h1 dir="ltr"><strong>Overview</strong></h1>

<p dir="ltr"><strong>Welcome to Unit 2, Introduction to Computer Programming, discussing&nbsp;computer science principles!&nbsp; The first lesson in this Unit&nbsp;is about the importance of variables and how they are used to store information about your programs.&nbsp; Students will learn&nbsp;how concatenation works within Javascript as well as how to put&nbsp;together string values in Javascript. &nbsp;This lesson ends&nbsp;with a discussion on how variables work in Javascript.&nbsp;&nbsp;</strong></p>

<p>&nbsp;</p>

<h1 dir="ltr"><strong>Purpose</strong></h1>

<p dir="ltr"><strong>This activity plants the initial seeds of syntax and storing values in memory within Javascript.&nbsp; Here is a good visual example of how variables are similar to storage containers. <img src="https://lh3.googleusercontent.com/je58xEAVFDM8iZcCgWFYi-g5e2e0qyHzakkTc1VQ0y5S4rSb7wUwGtDVzIXBRA6lWUSBF3YWnpVHEMbjvQ7Xb6QDvcPFE0rRcsWd1UTb1qXSNEqywTXKTSQVPn3mGiKlGu-Jj6q1" style="height:259px; width:624px" /></strong></p>

<p dir="ltr"><strong>Var bob = “Bob”;</strong></p>

<p dir="ltr"><strong>Var bool = true;</strong></p>

<p dir="ltr"><strong>Var thirtyFive = 35;</strong></p>

<p>&nbsp;</p>

<p dir="ltr"><strong>The above code snippets correspond to the images given.&nbsp; Each variable name itself is a container/box with a label.&nbsp; That label is the name given on the left side of the equals sign.&nbsp;&nbsp;</strong></p>

<p>&nbsp;</p>

<p dir="ltr"><strong>By working on concatenation of variables, students will understand how a Javascripts compiler works.&nbsp; As well as how variables are stored and can be called.&nbsp; In early lessons, students can confuse the value of a variable with the actual variable's name.&nbsp; The concatenation exercise will help to distinguish between the two.&nbsp;&nbsp;</strong></p>

<h1 dir="ltr"><strong>Teaching Guide</strong></h1>

<p dir="ltr"><strong>Students will return to this lesson often, if not for the concatenation aspect of putting together strings&nbsp;then for the basic concept of storing variables.&nbsp; This lesson goes hand in hand with returns which will be introduced later in the course.&nbsp;&nbsp;</strong></p>

<p>&nbsp;</p>

<p dir="ltr"><u><strong>Before your class</strong></u></p>

<p dir="ltr"><strong>Be sure that you have printed copies of the assignment available for each student.&nbsp; Having a digital copy to display on a screen is also encouraged.&nbsp; Remember to review the following major points within the lesson and be ready to talk them over in detail:</strong></p>

<ul>
	<li dir="ltr">
	<p dir="ltr"><strong>Variables are digital containers for storing values</strong></p>
	</li>
	<li dir="ltr">
	<p dir="ltr"><strong>Mathematical equations can be saved into a single variable.&nbsp;&nbsp;</strong></p>
	</li>
	<li dir="ltr">
	<p dir="ltr"><strong>Variable naming conventions</strong></p>
	<ul>
		<li dir="ltr">
		<p dir="ltr"><strong>Names can contain letters, digits, underscores, and dollar signs</strong></p>
		</li>
		<li dir="ltr">
		<p dir="ltr"><strong>Names must begin with a letter</strong></p>
		</li>
		<li dir="ltr">
		<p dir="ltr"><strong>Names can also begin with $ and _&nbsp;</strong></p>
		</li>
		<li dir="ltr">
		<p dir="ltr"><strong>Names are case sensitive (y and Y are different variables)</strong></p>
		</li>
		<li dir="ltr">
		<p dir="ltr"><strong>Reserved words (like JavaScript keywords) cannot be used as names</strong></p>
		</li>
		<li dir="ltr">
		<p dir="ltr"><strong>All variable names must be unique (no two alike)</strong></p>
		</li>
	</ul>
	</li>
	<li dir="ltr">
	<p dir="ltr"><strong>Assignment operator “=”&nbsp;</strong></p>
	</li>
</ul>

<p dir="ltr"><strong>Common errors :</strong></p>

<ul>
	<li dir="ltr">
	<p dir="ltr"><strong>Missing “+” operator between variables</strong></p>
	</li>
	<li dir="ltr">
	<p dir="ltr"><strong>No quotations around string values</strong></p>
	</li>
	<li dir="ltr">
	<p dir="ltr"><strong>Inserting&nbsp;a comma between large numbers such as 17,000 (In Javascript, you would just write the number as 17000 with no commas.)</strong></p>
	</li>
	<li dir="ltr">
	<p dir="ltr"><strong>Small misspellings on variable names</strong></p>
	</li>
	<li dir="ltr">
	<p dir="ltr"><strong>Syntax errors such as:&nbsp;</strong></p>
	<ul>
		<li dir="ltr">
		<p dir="ltr"><strong>Not using the var keyword</strong></p>
		</li>
		<li dir="ltr">

		</li>
	</ul>
	</li>
